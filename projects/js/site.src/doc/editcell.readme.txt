--плагин carret

function(act,selectionStart, selectionEnd)

act - указание на заказываемую операцию
selectionStart - начальная позиция
selectionEnd - конечная позиция

предназначен для работы с текстовым курсором в текстовых input и textarea. отлаживался для работы
в кооперации с плагином editcell.
Работа всегда ведется с первым из jQuery-коллекции объектов.

примеры использования
 - получить позицию курсора @return integer
 $('input').carret()
 $('input').carret('get')

 - получить позицию курсора, с учетом отмеченного текста @return array
 $('input').carret('sel')
 возвращемое значение массив из начальной позиции отметки и конечной позиции.

 - установить позицию курсора в текстовом поле.
 $('input').carret('set',3)
 $('input').carret(3)
 $('input').carret('set',3,4) - отметить кусок с 3 по 4 позиции

 - проверить, что указанная позиция внутри курсора. @return boolean
 $('input').carret('is',0) - курсор в начале поля ввода
 $('input').carret('is',$('input').val().length) - курсор в конце поля ввода
