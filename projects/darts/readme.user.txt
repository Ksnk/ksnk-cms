Система авторизации на сайте.
 Декларация:
 -- аккаунт на сайте имеет общие права, права на созданные клубы и права предоставленные
    по инвайтам.
 -- аккаунты делятся на общие, персональные, персональный инвайт и общий вход.
 -- Общий аккаунт предназначен для регистрации на компьютере, с которого вводятся результаты игр.
    Он имеет право создать один клуб или присоединится к одному клубу, разослать персональные ссылки
    создавать инвайт-ссылки
 -- персональный аккаунт имеет дополнительные (не обязательные) имя-фамилию. Аккаунт имеет право создать
    несколько клубов. При присоединении к клубу появляется возможность "подклеить" историю игр одного
    из игроков клуба с этим аккаунтом.

 -- персональный инвайт добавляет права на клуб к аккаунту. Для того, чтобы воспользоваться инвайтом,
    нужно войти на сайт под своим входом и перейти по инвайт-ссылке. При этом аккаунт "подклеивается"
    к игроку клуба, обозначенному в ссылке.

 -- общий инвайт служит для автоматического входа обладателя инвайта на сайт и просмотр результатов
    турнира. При подклеивании прав на клуб по этой ссылке действует ограничение времени на этот инвайт.

Реализация:

-- аккаунт может иметь поля : имя, пароль, expired и права

Отличия общего и персонального ака - наличие прав на RIGHT_CHANGE_USER.
-- сразу после регистрации появляется персональный акк:
   name, password, created, right:{*=RIGHT_CREATE_CLUB+RIGHT_CHANGE_USER+RIGHT_CHANGE_PASSWORD}
-- можно поменять акк на общий, исключив право *:RIGHT_CHANGE_USER. Обратно это
   право может включить только администратор сайта.

-- инвайты отличаются отсутствием прав на *:RIGHT_CHANGE_PASSWORD. Все инвайты не имеют пароля.
-- при вводе ссылки с существующим аком, права на клуб копируются в массив прав ака.
-- персональный инвайт, в дополнение к общему, содержить ID клуба, ID пользователя, при
   использовании персонального инвайта создается запись склейки.

-- при использовании инвайтов для входа без аков, разницы между ними нет.
